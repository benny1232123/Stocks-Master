- **修复**: 新增 `_fetch_shibor_overnight()` 多源回退函数（4 层降级）：
  1. 主源：`ak.rate_interbank(...)` 
  2. 备源1：`ak.shibor_data()` 取最新隔夜列
  3. 备源2：`ak.shibor_quote_history(symbol="隔夜")` 取最新价
  4. 兜底：读昨日 `macro_snapshot_{昨天}.pkl` 缓存
- **效果**: 即使当天所有 API 都挂，也用昨日数据填充；除非从未成功获取过才显示 "--"
- **文件**: `smcore/dashboard.py`；补入 `timedelta` 导入；py_compile 通过 ✅
