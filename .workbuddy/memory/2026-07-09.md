# 2026-07-09 工作日志

## GitHub Actions git push 冲突修复
- **问题**：`daily-pick.yml` 的"提交缓存到仓库"步骤 `git push` 被拒（remote contains work that you do not have locally）
- **原因**：Action 运行期间用户手动 push 到仓库，checkout 落后于远程
- **修复**：在 `git push` 前加 `git pull --rebase --autostash origin main || true`
- **文件**：`.github/workflows/daily-pick.yml` 第 149~163 行
